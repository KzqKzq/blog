# 拟物化风格个人博客开发需求文档

## 1. 项目概述
本项目旨在开发一个极具**拟物化（Skeuomorphic）**风格的个人博客网站。核心设计理念是“数字空间中的精致生活”，重现物理世界的触感与精致度。

**技术栈要求：**
- **核心框架**: React + Vite
- **UI 组件库**: [`@kzqkzq/tactile-ui`](https://kzqkzq.github.io/tactile-ui/components/index.html) (必须优先使用)
- **样式方案**: Vanilla CSS 或 TailwindCSS (配合自定义拟物化插件)
- **内容数据**: Markdown / MDX
- **部署**: 静态站点 (Static Site Generation)

## 2. 设计灵感与视觉风格

### 2.1 设计灵感来源：YYsuni.com
本项目的视觉设计借鉴了 [yysuni.com](https://www.yysuni.com/) 的优雅设计语言，并将其与现代拟物化风格相融合。

#### 核心色彩体系 (Color System)
| 角色 | 色值 | 用途 |
|------|------|------|
| **品牌主色 (Brand)** | `#de4331` | 强调色、logo、重要按钮、链接悬停 |
| **品牌辅色 (Brand Secondary)** | `#FCC841` | 装饰性元素、徽章、高亮标签 |
| **主文字色 (Primary)** | `#4E3F42` | 标题、正文、主要内容 |
| **次文字色 (Secondary)** | `#7b888e` | 辅助文字、描述、时间戳 |
| **背景色 (Background)** | `#eeeeee` | 页面主背景（配合拟物化纹理叠加） |
| **边框色 (Border)** | `#ffffff` | 卡片边框、分割线 |
| **卡片背景 (Card)** | `rgba(255,255,255,0.4)` | 半透明卡片，配合磨砂玻璃效果 |
| **文章背景 (Article)** | `rgba(255,255,255,0.8)` | 内容区域背景 |

#### 布局与空间设计 (Layout & Spacing)
从 YYsuni.com 借鉴的布局原则：
- **居中内容容器**: 内容区域宽度约 `800-1000px`，居中展示
- **充足的呼吸空间**: 大量留白，段落间距宽敞
- **弹性卡片布局**: 使用 Flexbox 和 Grid 实现自适应卡片排列
- **模糊背景效果**: 背景使用 `filter: blur()` 创建层次感
- **响应式设计**: 移动端采用垂直堆叠布局，间距适当调整

#### 排版体系 (Typography)
- **特色字体**: 使用 `Averia Gruesa Libre` 作为装饰性标题字体，带有手写和复古质感
- **主字体栈**: `Inter`, `System UI`, `Roboto` 等现代无衬线字体作为正文
- **中文字体**: `PingFang SC`, `Microsoft YaHei`, `Noto Sans SC`
- **等宽字体**: `JetBrains Mono`, `Fira Code` 用于代码展示

#### 交互细节 (Micro-interactions)
- **自定义光标**: 使用 SVG 自定义鼠标指针，增加品牌识别
- **平滑过渡**: 所有状态变化使用 `transition` 动画，时长 `0.2s-0.3s`
- **卡片悬停**: 鼠标悬停时卡片有微妙的上浮和阴影变化
- **链接样式**: 悬停时变为品牌主色 `#de4331`

---

### 2.2 现代新拟物主义设计哲学 (Modern Neumorphism)
本项目追求的是**现代新拟物主义**——不是 iOS 6 时代的完全拟物化，而是在**极简扁平设计**基础上，通过**微妙的深度、柔和的光影、和精致的触感反馈**，让界面"活"起来。

> **核心理念**: 90% 抽象极简 + 10% 拟物触感 = 现代精致

#### 设计原则

| 原则 | ❌ 避免（旧拟物） | ✅ 采用（新拟物） |
|------|------------------|------------------|
| **材质** | 完整的木纹/皮革纹理 | 微妙的渐变暗示材质，磨砂玻璃效果 |
| **阴影** | 生硬的 drop-shadow | 柔和的多层阴影，凹凸感（inset shadow） |
| **图标** | 3D 写实图标 | 简洁线性图标 + 微妙的立体感 |
| **色彩** | 浓重的材质色 | 干净的品牌色 + 柔和中性色背景 |
| **交互** | 厚重的按压反馈 | 轻盈的弹性动画（spring animation） |

#### 关键视觉特征

1. **Soft UI / 柔性界面**
   - 元素像是从背景中"浮起"或"凹陷"，而非放置在上面
   - 使用 `box-shadow` 的光影对比：左上高光 + 右下阴影
   - 背景色与元素色为同一色系，仅靠阴影区分层级

2. **Glassmorphism / 玻璃态**
   - 半透明背景 `background: rgba(255,255,255,0.4)`
   - 背景模糊 `backdrop-filter: blur(10px)`
   - 微妙的边框高光 `border: 1px solid rgba(255,255,255,0.2)`

3. **微材质暗示**
   - 用极淡的噪点（noise texture）暗示纸张质感
   - 用微妙的渐变暗示金属光泽
   - **不使用**真实的木纹、皮革图片

4. **精致的微交互**
   - 按钮按压：轻微的 `scale(0.98)` + 阴影收缩
   - 卡片悬停：轻微上浮 `translateY(-2px)` + 阴影扩展
   - 开关切换：弹性动画 `cubic-bezier(0.68, -0.55, 0.265, 1.55)`

#### CSS 设计 Token 示例
```css
:root {
  /* Neumorphism 阴影 */
  --shadow-raised: 6px 6px 12px rgba(0,0,0,0.08), -6px -6px 12px rgba(255,255,255,0.8);
  --shadow-pressed: inset 4px 4px 8px rgba(0,0,0,0.06), inset -4px -4px 8px rgba(255,255,255,0.6);
  
  /* 过渡动画 */
  --transition-base: 0.2s ease-out;
  --transition-spring: 0.4s cubic-bezier(0.34, 1.56, 0.64, 1);
  
  /* 圆角 */
  --radius-sm: 8px;
  --radius-md: 16px;
  --radius-lg: 24px;
}
```

## 3. 页面功能与设计构想

### 3.1 首页：个人工作空间 (Personal Workspace)
首页是一个**精致的个人空间**，不是传统的列表 feed，而是有层次感和呼吸感的布局。

*   **布局**: 浅色中性背景（`#eeeeee`），居中的内容容器，大量留白
*   **主视觉**: 个人介绍卡片（头像 + 简介 + 社交链接），使用磨砂玻璃效果
*   **导航项**: 使用柔和的卡片形式，hover 时有微妙的上浮动画：
    *   **文章列表**: 带有品牌色点缀的内容卡片
    *   **关于我**: 个人档案卡片，圆角 + 柔和阴影
    *   **搜索**: 带有放大镜图标的输入框组件
    *   **设置/主题**: 优雅的开关组件（新拟物风格的凹凸感）
    *   **社交链接**: 图标按钮组，hover 时变为品牌色
*   **微交互**: 卡片悬停时 `translateY(-2px)` + 阴影扩展；点击时轻微 `scale(0.98)`

### 3.2 内容列表：分区的视觉差异
针对不同内容类型，采用不同的卡片样式和视觉暗示。**所有板块都支持 Markdown 渲染**。

#### 3.2.1 技术博客 (Tech Blog)
展示严肃的技术文章与教程。
*   **视觉风格**: 
    *   卡片背景稍显"冷"色调，暗示专业感
    *   标题使用等宽字体（Monospace）暗示代码/技术
    *   带有标签（Tags）用于分类（React, CSS, Backend 等）
*   **列表项设计**:
    *   干净的卡片布局，左侧日期 + 右侧标题描述
    *   代码片段预览使用深色代码主题区块
    *   可带有"Featured"或"Series"等微妙标签

#### 3.2.2 项目展示 (Projects)
展示个人作品与开源项目。
*   **视觉风格**: 
    *   使用 Grid 卡片布局，强调视觉冲击力
    *   每个项目卡片可带有项目图标或缩略图
    *   悬停时卡片有更明显的上浮效果
*   **列表项设计**:
    *   项目名称 + 简介 + 技术栈标签
    *   GitHub Star 等数据使用小徽章展示
    *   链接按钮（Demo / Source）使用品牌色

#### 3.2.3 随笔 (Essays)
展示生活感悟与非技术类文章。
*   **视觉风格**: 
    *   卡片背景稍显"暖"色调，暗示轻松感
    *   字体偏向衬线体，排列更自由灵动
    *   可配有少量装饰元素（如日期戳、引用块）
*   **列表项设计**:
    *   图文混排，图片使用圆角处理
    *   日期以手写风格或更随意的格式展示
    *   更注重文字内容的呼吸感

### 3.3 文章详情 (Reader)
*   **核心功能**: 将 Markdown 渲染为精美的阅读体验
*   **视觉风格**: 
    *   干净的白色/米色阅读区域，减少眼睛疲劳
    *   最大宽度 `680px`，保证阅读舒适度
    *   衬线体（Serif）作为标题，无衬线体作为正文
*   **功能组件**:
    *   **代码块**: 现代深色代码主题，带有语言标签和复制按钮
    *   **图片**: 居中展示，带有微妙的阴影，可点击放大
    *   **引用块**: 左侧品牌色边框，背景使用浅灰色
    *   **目录 (TOC)**: 侧边固定，使用透明玻璃态背景

### 3.4 关于我 (About)
*   设计成一个精致的个人档案卡片
*   头像使用圆形 + 微妙边框 + 阴影
*   技能使用进度条或标签云展示
*   工作经历使用时间线组件

## 4. 组件开发指南
请优先使用 `@kzqkzq/tactile-ui` 中的组件。如果遇到需求无法满足，请**构建新拟物风格组件**，遵循以下规范：

1.  **视觉层级**:
    *   底层：统一的背景色调（浅灰 `#eeeeee` 或浅奶白）
    *   中层：元素通过 Neumorphism 阴影"浮起"或"凹陷"
    *   顶层：品牌色点缀、hover 状态高光

2.  **通用组件规范**:
    *   **Button**: 柔和圆角，hover 时阴影扩展，active 时 `scale(0.98)` + 凹陷阴影
    *   **Toggle**: 新拟物凹槽 + 浮起的圆形手柄，弹性动画切换
    *   **Input**: 凹陷的输入区域（inset shadow），聚焦时品牌色边框
    *   **Card**: 浮起的圆角容器，hover 时微妙上浮

3.  **动画规范**:
    *   使用弹性曲线 `cubic-bezier(0.34, 1.56, 0.64, 1)` 模拟物理回弹
    *   过渡时长控制在 `0.2s - 0.4s`，不宜过慢
    *   避免过于夸张的动画，保持微妙精致

## 5. 给 AI 的开发提示词 (Prompt)
> "你是一个精通**现代新拟物主义 (Neumorphism)** UI 设计的前端专家。请使用 React 和 @kzqkzq/tactile-ui 组件库构建这个博客。
> 
> 设计原则：**90% 极简抽象 + 10% 拟物触感**。不要使用真实的木纹、皮革纹理图片。
> 
> CSS 技巧：
> - 使用双层 `box-shadow`（左上亮 + 右下暗）创建柔和凸起效果
> - 使用 `inset box-shadow` 创建凹陷效果
> - 使用 `backdrop-filter: blur()` 和半透明背景创建玻璃态
> - 使用 `cubic-bezier` 弹性曲线创建有触感的动画
> 
> 不要：3D 写实图标、厚重材质、老式终端样式、拍立得/便利贴等完全拟物效果。
> 要：干净留白、柔和阴影、微妙立体感、品牌色点缀、精致微交互。"
